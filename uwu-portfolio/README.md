# 💖 ANKUR's Kawaii Portfolio ✨

A super cute, pastel-themed portfolio built with React, Tailwind CSS, and Framer Motion! 🌸

## 🌈 Features

- 🎀 **Kawaii Design**: Soft pastels, cute emojis, and adorable animations
- 🌙 **Day/Night Mode**: Toggle between sunny pastels and starry night themes
- ✨ **Smooth Animations**: Framer Motion powered interactions
- 💖 **Custom Cursor**: Heart cursor for extra cuteness
- 🧋 **Floating Elements**: Animated background decorations
- 📱 **Responsive**: Looks great on all devices

## 🛠️ Tech Stack

- **React** + **Vite** - Fast development and building
- **Tailwind CSS** - Utility-first styling with custom kawaii colors
- **Framer Motion** - Smooth animations and transitions
- **Lucide React** - Beautiful icons

## 🚀 Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

## 🎨 Customization

### Colors
The kawaii color palette is defined in `tailwind.config.js`:
- `kawaii-pink`: #FFC7C7
- `kawaii-blue`: #C8E7FF  
- `kawaii-lavender`: #E4D9FF
- `kawaii-cream`: #FFF9F0

### Fonts
- **Headings**: Fredoka One (kawaii style)
- **Body**: Quicksand (cute and readable)

### Components
- `Hero.jsx` - Landing section with animated avatar
- `About.jsx` - Personal info with flip cards
- `Projects.jsx` - Project showcase with hover effects
- `Contact.jsx` - Contact form and social links
- `FloatingElements.jsx` - Animated background decorations

## 🌸 Deployment

Ready to deploy on Vercel, Netlify, or any static hosting service!

```bash
npm run build
```
